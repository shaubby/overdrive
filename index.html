why gamemaker?

gamemaker is a super easy to use game development software that lets you make games quickly and easily. it's perfect for beginners and experienced developers alike!

with gamemaker, you can create 2d games with drag-and-drop functionality or by using its built-in scripting language, gml. it's a great way to learn the basics of game development and get started on your own projects!
